# Smart-Random-Picker (课堂智能随机点名系统)

![Version](https://img.shields.io/badge/Version-v2.8.1-blue.svg)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey.svg)
![Python](https://img.shields.io/badge/Python-3.8+-green.svg)

Smart-Random-Picker 是一款专为教育教学环境设计的轻量级随机点名工具。本项目致力于解决传统点名软件在老旧多媒体教学设备（如 Windows 7 系统）上的兼容性痛点，提供开箱即用、防崩溃且具备高度定制化能力的点名体验。

---

## 🌟 核心特性

* **极致兼容与防崩溃引擎**：内置“智能双模文本解码器”（支持 UTF-8-SIG 与 GBK 无缝回退机制），彻底根除在老旧 Windows 系统中因新建 TXT 默认编码导致的 `UnicodeDecodeError` 闪退问题。剥离了重量级图像渲染库，确保在低配置白板上“秒级启动”。
* **静默热重载技术**：支持在不重启软件的情况下，通过点击“开始”按钮隐性重载数据，或使用“刷新”按钮显性校验数据，实现名单的即改即用。
* **暗箱拦截机制（黑名单）**：内置隐藏的管理员级控制后台，可设定不参与抽签的人员名单，整个拦截过程对前端用户完全透明。
* **OTA 在线更新检测**：启动时自动比对远端仓库版本号，并在发现新版本时提供平滑的更新跳转引导。
* **用户反馈闭环**：主界面集成反馈入口，一键跳转至 GitHub Issues 页面，便于收集终端用户需求。

## 🚀 快速上手 (终端用户指南)

### 1. 软件安装与准备
本软件为单文件便携版 (Standalone Executable)，**无需安装**。
1. 前往 [Releases 页面](../../releases/latest) 下载最新的 `SmartPicker.exe` 文件。
2. 将下载的 `.exe` 文件放置在一个新建的专属文件夹中。

### 2. 名单配置
在软件同级目录下，新建一个文本文档并命名为 **`名单.txt`**。
* 每行输入一个姓名。
* 编码格式不受限制（软件支持智能识别）。

### 3. 基本操作
* **抽取人数调节**：拖动主界面的滑块，支持单次抽取 1~20 人。
* **开始/停止**：点击开始按钮屏幕滚动姓名，点击停止展示最终结果。
* **抽取历史**：界面下方实时记录每一次抽取的历史名单与时间戳，防止遗忘。

## ⚙️ 高阶功能：管理员暗箱操作 (黑名单机制)

对于请假、已点名或特殊情况需要避开抽签的学生，系统提供了隐蔽的拦截机制：

1. **创建名单**：在软件同级目录下，新建 **`黑名单.txt`**，每行输入一个需要跳过的姓名。
2. **触发后台**：运行软件后，**双击主界面顶部的标题文字**（“课堂随机点名”）。
3. **身份验证**：在弹出的验证窗口中输入管理员密码（默认密码：`114514`）。
4. **生效确认**：验证通过后，系统将提示“暗箱操作已就绪”，此时黑名单中的人员将在后续的抽取中被完全隔离。

## 🛠️ 开发者构建指南

如果您希望基于源码进行二次开发，请参考以下步骤：

### 环境依赖
建议使用 **Python 3.8.10** 以确保编译产物向下兼容 Windows 7。
```bash
pip install pygame pyinstaller
```

### 极速编译命令
使用 PyInstaller 进行单文件打包，并注入所有隐藏依赖组件：
```bash
pyinstaller --noconsole --onefile --clean --name="SmartPicker_V2.8.1_Real" --hidden-import=pygame dianming.py
```
